# AI Tattoo Designer

## Overview

AI Tattoo Designer is an AI-powered web application that generates personalized tattoo concepts using Google's Gemini AI.

Users upload an image of the desired body area and describe their preferred tattoo style using keywords. The application analyzes the uploaded image and generates tattoo ideas that match the user's preferences.

The project explores the use of Generative AI for personalized tattoo design.

---

## Features

- Upload body images
- AI-powered image understanding
- Keyword-based tattoo generation
- Personalized tattoo suggestions
- Interactive user interface

---

## Technologies

- React
- TypeScript
- Google Gemini API
- HTML
- CSS

---

## How It Works

1. Upload an image.
2. Enter keywords describing the desired tattoo.
3. The image and prompt are sent to Gemini AI.
4. Gemini analyzes the image and generates tattoo design suggestions.
